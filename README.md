# Rakta-Seva Connect

## Project Description
Rakta-Seva Connect is an Android application developed to help users find blood donors and request blood during emergencies. The app connects donors and receivers quickly and efficiently.

## Features
- User Registration and Login
- Search Blood Donors
- Emergency Blood Request
- Donor Details Management
- Blood Group Filtering
- User-Friendly Interface

## Technologies Used
- Java / Kotlin
- Android Studio
- Firebase Authentication
- Firebase Realtime Database
- XML

## Modules
1. User Authentication
2. Donor Registration
3. Blood Request System
4. Search and Filter
5. Database Management

## Objectives
- To reduce delay in finding blood donors
- To provide emergency blood support
- To create a digital blood donor network


## Installation Steps
1. Clone the repository
2. Open in Android Studio
3. Connect Firebase
4. Run the project on emulator or mobile
